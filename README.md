## 실행방법
<br/>

### in terminal

1. ```
    git clone https://github.com/Wuyuh-sun/real_time_currency_converter.git
    ```
2. ```
    npm i
    ```
3. ```
    npm run start
    ```
4. http://localhost:3000/
